Documentação oficial da API PAYPAY Angola para integração de pagamentos. Este projeto fornece uma interface amigável e interativa para desenvolvedores que desejam integrar soluções de pagamento em suas aplicações.

## 🚀 Demonstração

Acesse a documentação online: [https://paypay-ao-api.netlify.app](https://paypay-ao-api.netlify.app)

## 📋 Funcionalidades

- ✅ **8 endpoints disponíveis** para diferentes métodos de pagamento
- ✅ **Tutoriais interativos** passo a passo
- ✅ **Exemplos práticos** de requisições e respostas
- ✅ **Interface responsiva** para todos os dispositivos
- ✅ **Códigos de retorno** documentados
- ✅ **Seção de segurança** com informações sobre criptografia

## 🔧 Endpoints Implementados

| ID | Método | Endpoint | Descrição |
|----|--------|----------|-----------|
| 3.1 | `POST` | `/payment/app` | Cria pagamento via app PAYPAY |
| 3.2 | `POST` | `/payment/multicaixa` | Pagamento via MULTICAIXA Expre |
| 3.3 | `POST` | `/payment/reference` | Gera referência para pagamento |
| 3.4 | `POST` | `/refund` | Solicita reembolso |
| 3.5 | `POST` | `/payment/close` | Fecha pagamento pendente |
| 3.6 | `GET` | `/query/status` | Consulta status da transação |
| 3.7 | `POST` | `/transfer/bank` | Transferência bancária |
| 3.8 | `POST` | `/transfer/account` | Transferência entre contas |
