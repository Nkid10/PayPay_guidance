Pagamento via App PAYPAY

Cria um pagamento para ser pago através do aplicativo PAYPAY

POST https://api.paypayafrica.com/v1/payment/app
QR Code URL Scheme Mobile

Descrição

Este endpoint cria uma solicitação de pagamento que será processada através do aplicativo PAYPAY. O cliente pode pagar escaneando um QR Code ou através de um link profundo que abre o app.

Parâmetros da Requisição

Parâmetro Obrig. Tipo Descrição
request_no Sim String(6,32) Número único da requisição (alfanumérico e underline)
service Sim String Valor fixo: "instant_trade"
version Sim String Versão da API: "1.0"
partner_id Sim String(32) Número do merchant (obtido no portal)
charset Sim String "UTF-8"
sign Sim String Valor da assinatura RSA
sign_type Sim String "RSA"
timestamp Sim String(19) Formato "yyyy-MM-dd HH:mm:ss" (GMT+1)
format Sim String "JSON"
language Sim String "pt" ou "en"
biz_content Sim String Conteúdo de negócio CRIPTOGRAFADO

Conteúdo de Negócio (biz_content)

Parâmetro Obrig. Tipo Descrição
payer_ip Sim String IP público do comprador
sale_product_code Sim String(9) Contactar suporte PAYPAY
cashier_type Sim String "SDK"
timeout_express Não String(6) Tempo de expiração (ex: 15m, 2H). Padrão: 2H
return_url Não String URL de retorno após pagamento

Informações do Produto (trade_info)

Parâmetro Obrig. Tipo Descrição
out_trade_no Sim String Número do pedido do merchant
subject Sim String Nome do produto/serviço
currency Não String "AOA" (padrão)
price Sim String Preço unitário
quantity Sim String Quantidade
total_amount Sim String Valor total (price * quantity)
payee_identity Sim String Número do merchant
payee_identity_type Não String Tipo de identidade (1 - member ID, 2 - login)

Exemplo com JavaScript (Fetch) 
// Exemplo com JavaScript (Node.js ou Browser)
const fetch = require('node-fetch'); // Para Node.js

const url = 'https://api.paypayafrica.com/v1/payment/app';
const data = {
    request_no: '20240320123456789',
    service: 'instant_trade',
    version: '1.0',
    partner_id: '200001835716',
    charset: 'UTF-8',
    sign: 'N6EPWFhOOiqP7PfgEWgffszU6SCMUimc4pQv4XL...',
    sign_type: 'RSA',
    timestamp: '2024-03-20 14:30:00',
    format: 'JSON',
    language: 'pt',
    biz_content: 'GkTO5wRUFTij4GPniUQChPquaLoDNzi3z...'
};

fetch(url, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => {
    console.log('Resposta:', data);
    // Processar a resposta
    if (data.code === 'S0001') {
        const bizContent = JSON.parse(atob(data.biz_content)); // Decodificar base64
        console.log('Link dinâmico:', bizContent.dynamic_link);
        console.log('Token:', bizContent.trade_token);
    }
})
.catch(error => console.error('Erro:', error));
Exemplo com Python (requests) 
import requests
import json
import base64

url = "https://api.paypayafrica.com/v1/payment/app"

payload = {
    "request_no": "20240320123456789",
    "service": "instant_trade",
    "version": "1.0",
    "partner_id": "200001835716",
    "charset": "UTF-8",
    "sign": "N6EPWFhOOiqP7PfgEWgffszU6SCMUimc4pQv4XL...",
    "sign_type": "RSA",
    "timestamp": "2024-03-20 14:30:00",
    "format": "JSON",
    "language": "pt",
    "biz_content": "GkTO5wRUFTij4GPniUQChPquaLoDNzi3z..."
}

headers = {
    'Content-Type': 'application/json'
}

try:
    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    
    data = response.json()
    print("Resposta:", json.dumps(data, indent=2))
    
    # Processar a resposta
    if data.get('code') == 'S0001':
        # Decodificar biz_content (assumindo base64)
        biz_content = json.loads(base64.b64decode(data['biz_content']))
        print("Link dinâmico:", biz_content['dynamic_link'])
        print("Token:", biz_content['trade_token'])
        
except requests.exceptions.RequestException as e:
    print(f"Erro na requisição: {e}")
Exemplo com PHP (cURL) 
<?php
$url = 'https://api.paypayafrica.com/v1/payment/app';

$data = array(
    'request_no' => '20240320123456789',
    'service' => 'instant_trade',
    'version' => '1.0',
    'partner_id' => '200001835716',
    'charset' => 'UTF-8',
    'sign' => 'N6EPWFhOOiqP7PfgEWgffszU6SCMUimc4pQv4XL...',
    'sign_type' => 'RSA',
    'timestamp' => '2024-03-20 14:30:00',
    'format' => 'JSON',
    'language' => 'pt',
    'biz_content' => 'GkTO5wRUFTij4GPniUQChPquaLoDNzi3z...'
);

$json_data = json_encode($data);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, "POST");
curl_setopt($ch, CURLOPT_POSTFIELDS, $json_data);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, array(
    'Content-Type: application/json',
    'Content-Length: ' . strlen($json_data)
));

$response = curl_exec($ch);
$http_code = curl_getinfo($ch, CURLINFO_HTTP_CODE);

if (curl_errno($ch)) {
    echo 'Erro cURL: ' . curl_error($ch);
} else {
    echo "HTTP Code: $http_code\n";
    $response_data = json_decode($response, true);
    echo "Resposta: " . json_encode($response_data, JSON_PRETTY_PRINT) . "\n";
    
    // Processar a resposta
    if ($response_data['code'] == 'S0001') {
        // Decodificar biz_content (assumindo base64)
        $biz_content = json_decode(base64_decode($response_data['biz_content']), true);
        echo "Link dinâmico: " . $biz_content['dynamic_link'] . "\n";
        echo "Token: " . $biz_content['trade_token'] . "\n";
    }
}

curl_close($ch);
?>

Descrição dos campos:

  • out_trade_no: Número do pedido do merchant
  • trade_no: Número do pedido PAYPAY
  • status: Status do pagamento (S-Sucesso, P-Processando, F-Falha)
  • trade_token: Token do pagamento para URL scheme
  • dynamic_link: Link dinâmico (pode ser usado para gerar QR Code)

Códigos de Status

Status Descrição
P Pagamento em processamento (Payment processing)
S Pagamento concluído com sucesso (Payment successful)
F Falha no pagamento (Payment failed)

Sugestões de Implementação

Para aplicações mobile: Utilize o URL scheme paypayao://trade/pay?action=pay&tradeToken={trade_token} para abrir diretamente o app PAYPAY.

Para aplicações web/desktop: Converta o dynamic_link em um QR Code para que o usuário possa escanear com o app PAYPAY.

Exemplo de geração de QR Code:

// JavaScript
const QRCode = require('qrcode');
QRCode.toFile('qrcode.png', dynamic_link);
