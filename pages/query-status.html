Consultar Status de Transação

Consulta o status atual de qualquer transação (pagamento, transferência bancária, transferência interna)

POST https://api.paypayafrica.com/v1/query/status
Consulta Status Verificação

Descrição

Este endpoint permite consultar o status detalhado de qualquer transação processada pela PAYPAY, incluindo pagamentos, reembolsos e transferências. É essencial para sincronizar o estado das transações entre o sistema do merchant e a PAYPAY.

Parâmetros da Requisição

Parâmetro Obrig. Tipo Descrição
request_no Sim String(6,32) Número único da requisição (alfanumérico e underline)
service Sim String Valor fixo: "trade_query"
version Sim String Versão da API: "1.0"
partner_id Sim String(32) Número do merchant (obtido no portal)
charset Sim String "UTF-8"
sign Sim String Valor da assinatura RSA
sign_type Sim String "RSA"
timestamp Sim String(19) Formato "yyyy-MM-dd HH:mm:ss" (GMT+1)
format Sim String "JSON"
language Sim String "pt" ou "en"
biz_content Sim String Conteúdo de negócio CRIPTOGRAFADO

Conteúdo de Negócio (biz_content)

Parâmetro Obrig. Tipo Descrição
out_trade_no Sim String(6,32) Número do pedido do merchant a ser consultado

Exemplo com Python 
import requests
import json
import base64
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_v1_5

# Dados do biz_content
biz_content = {
    "out_trade_no": "20240320123456789"
}

# Função para criptografar (exemplo)
def encrypt_biz_content(data):
    # Implementar criptografia RSA com chave pública
    # Por simplicidade, este exemplo apenas converte para base64
    json_str = json.dumps(data)
    return base64.b64encode(json_str.encode()).decode()

# Requisição
url = "https://api.paypayafrica.com/v1/query/status"

payload = {
    "request_no": "20240320123456789",
    "service": "trade_query",
    "version": "1.0",
    "partner_id": "200001835716",
    "charset": "UTF-8",
    "sign": "N6EPWFhOOiqP7PfgEWgffszU6SCMUimc4pQv4XL...",
    "sign_type": "RSA",
    "timestamp": "2024-03-20 14:30:00",
    "format": "JSON",
    "language": "pt",
    "biz_content": encrypt_biz_content(biz_content)
}

headers = {
    'Content-Type': 'application/json'
}

try:
    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    
    data = response.json()
    print("Resposta:", json.dumps(data, indent=2))
    
    if data.get('code') == 'S0001':
        # Decodificar resposta
        response_biz = json.loads(base64.b64decode(data['biz_content']))
        print(f"Status: {response_biz['status']}")
        print(f"Valor: {response_biz['amount']} AOA")
        print(f"Data: {response_biz['modify_time']}")
        
        # Verificar se o pagamento foi bem-sucedido
        if response_biz['status'] in ['TRADE_SUCCESS', 'TRADE_FINISHED']:
            print("✅ Pagamento confirmado!")
            
except requests.exceptions.RequestException as e:
    print(f"Erro na requisição: {e}")

<?php
// Dados do biz_content
$bizContent = [

// Função para criptografar (exemplo)
function encryptBizContent($data) {
    $json = json_encode($data);
    return base64_encode($json);
}

$url = 'https://api.paypayafrica.com/v1/query/status';

$data = [
    'request_no' => '20240320123456789',

    'sign_type' => 'RSA',
    'timestamp' => '2024-03-20 14:30:00',
    'format' => 'JSON',
    'language' => 'pt',
    'biz_content' => encryptBizContent($bizContent)
];

$json_data = json_encode($data);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, "POST");
curl_setopt($ch, CURLOPT_POSTFIELDS, $json_data);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json',
    'Content-Length: ' . strlen($json_data)
]);

$response = curl_exec($ch);
$http_code = curl_getinfo($ch, CURLINFO_HTTP_CODE);

if (curl_errno($ch)) {
    echo 'Erro cURL: ' . curl_error($ch);
} else {
    echo "HTTP Code: $http_code\n";
    $response_data = json_decode($response, true);
    echo "Resposta: " . json_encode($response_data, JSON_PRETTY_PRINT) . "\n";
    
    if ($response_data['code'] == 'S0001') {
        // Decodificar biz_content da resposta
        $responseBiz = json_decode(base64_decode($response_data['biz_content']), true);
        echo "Status: " . $responseBiz['status'] . "\n";
        echo "Valor: " . $responseBiz['amount'] . " AOA\n";
        echo "Data: " . $responseBiz['modify_time'] . "\n";
        
        // Verificar se o pagamento foi bem-sucedido
        if (in_array($responseBiz['status'], ['TRADE_SUCCESS', 'TRADE_FINISHED'])) {
            echo "✅ Pagamento confirmado!\n";
        }
    }
}

curl_close($ch);
?>

Tabela de Status

Status Tipo Descrição Ação Recomendada
WAIT_BUYER_PAY Pagamento Aguardando pagamento do comprador Aguardar ou fechar se expirado
TRADE_CLOSED Pagamento Pagamento fechado/cancelado Não retentar, iniciar novo pedido
TRADE_SUCCESS Pagamento Pagamento bem-sucedido Liberar produto/serviço
TRADE_FINISHED Pagamento Pagamento bem-sucedido e liquidado Liberar produto/serviço
REFUND_SUCCESS Reembolso Reembolso realizado com sucesso Registrar estorno no sistema
REFUND_FAIL Reembolso Falha no reembolso Verificar motivo e retentar
TRANSFER_SUCCESS Transferência Transferência bem-sucedida Registrar conclusão
TRANSFER_FAIL Transferência Falha na transferência Verificar motivo e retentar

Nota: TRADE_SUCCESS e TRADE_FINISHED podem ser tratados como pagamento confirmado para liberação de produtos/serviços.

🔍 Boas Práticas

  • Consultar antes de retentar: Sempre consulte o status antes de retentar operações sensíveis para evitar duplicidade
  • Polling inteligente: Para transações em processamento, implemente consultas periódicas com intervalo crescente (ex: 5s, 10s, 30s, 60s)
  • Timeout com verificação: Implemente timeout global com consulta de verificação final para garantir que a operação foi concluída
  • Status de sucesso: Considere TRADE_SUCCESS e TRADE_FINISHED como indicadores de sucesso no pagamento
  • Cache de resultados: Armazene resultados de consulta para reduzir chamadas desnecessárias

Verificação após pagamento

Lógica de verificação 
// Pseudocódigo para verificação de pagamento
function verificarPagamento(out_trade_no) {
    const maxTentativas = 10;
    const intervalo = 3000; // 3 segundos
    
    for (let tentativa = 1; tentativa <= maxTentativas; tentativa++) {
        const status = consultarStatus(out_trade_no);
        
        if (status === 'TRADE_SUCCESS' || status === 'TRADE_FINISHED') {
            return { sucesso: true, mensagem: 'Pagamento confirmado' };
        }
        
        if (status === 'TRADE_CLOSED') {
            return { sucesso: false, mensagem: 'Pagamento fechado/cancelado' };
        }
        
        if (status === 'WAIT_BUYER_PAY' && tentativa < maxTentativas) {
            aguardar(intervalo);
            continue;
        }
        
        break;
    }
    
    return { sucesso: false, mensagem: 'Timeout na verificação' };
}
