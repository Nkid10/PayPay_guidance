Reembolso de Pagamento

Solicita reembolso total ou parcial de uma transação (até 180 dias)

POST https://api.paypayafrica.com/v1/refund
180 dias Total/Parcial Reembolso

Descrição

Este endpoint permite solicitar reembolsos de transações. Suporta reembolsos totais ou parciais, com prazo máximo de 180 dias após a data da transação original. O status do reembolso pode ser consultado via API de consulta ou através de notificações assíncronas.

Parâmetros da Requisição

Parâmetro Obrig. Tipo Descrição
request_no Sim String(6,32) Número único da requisição (alfanumérico e underline)
service Sim String Valor fixo: "trade_refund"
version Sim String Versão da API: "1.0"
partner_id Sim String(32) Número do merchant (obtido no portal)
charset Sim String "UTF-8"
sign Sim String Valor da assinatura RSA
sign_type Sim String "RSA"
timestamp Sim String(19) Formato "yyyy-MM-dd HH:mm:ss" (GMT+1)
format Sim String "JSON"
language Sim String "pt" ou "en"
biz_content Sim String Conteúdo de negócio CRIPTOGRAFADO

Conteúdo de Negócio (biz_content)

Parâmetro Obrig. Tipo Descrição
orig_out_trade_no Sim String(32) Número do pedido original a ser reembolsado
out_trade_no Sim String(6,32) Novo número para o pedido de reembolso (deve ser único)
refund_amount Sim String(15) Valor a ser reembolsado (máx. 2 casas decimais)

Exemplo com Python 
import requests
import json
import base64
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_v1_5

# Dados do biz_content
biz_content = {
    "orig_out_trade_no": "20240320123456789",
    "out_trade_no": "20240320123456789ref1",
    "refund_amount": "50.00"
}

# Função para criptografar (exemplo)
def encrypt_biz_content(data):
    # Implementar criptografia RSA com chave pública
    # Por simplicidade, este exemplo apenas converte para base64
    json_str = json.dumps(data)
    return base64.b64encode(json_str.encode()).decode()

# Requisição
url = "https://api.paypayafrica.com/v1/refund"

payload = {
    "request_no": "20240320123456789",
    "service": "trade_refund",
    "version": "1.0",
    "partner_id": "200001835716",
    "charset": "UTF-8",
    "sign": "N6EPWFhOOiqP7PfgEWgffszU6SCMUimc4pQv4XL...",
    "sign_type": "RSA",
    "timestamp": "2024-03-20 14:30:00",
    "format": "JSON",
    "language": "pt",
    "biz_content": encrypt_biz_content(biz_content)
}

headers = {
    'Content-Type': 'application/json'
}

try:
    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    
    data = response.json()
    print("Resposta:", json.dumps(data, indent=2))
    
    if data.get('code') == 'S0001':
        # Decodificar resposta
        response_biz = json.loads(base64.b64decode(data['biz_content']))
        print(f"Status do reembolso: {response_biz['status']}")
        print(f"Número do reembolso: {response_biz['trade_no']}")
        
        if response_biz['status'] == 'S':
            print("✅ Reembolso processado com sucesso!")
        elif response_biz['status'] == 'P':
            print("⏳ Reembolso em processamento...")
        
except requests.exceptions.RequestException as e:
    print(f"Erro na requisição: {e}")

    }

Status de Reembolso

Status Descrição Próximos Passos
S Reembolso bem-sucedido (REFUND_SUCCESS) Registrar estorno no sistema do merchant
P Reembolso em processamento Aguardar notificação ou consultar status
F Falha no reembolso (REFUND_FAIL) Verificar motivo e retentar se possível

⚠️ Regras de Reembolso

  • Reembolsos podem ser totais ou parciais - O valor não pode exceder o valor original
  • Prazo máximo: 180 dias após a transação original
  • out_trade_no único: Cada solicitação de reembolso deve ter um número de pedido único
  • Status final: Deve ser confirmado via consulta ou notificação assíncrona
  • Reembolsos parciais: Múltiplos reembolsos parciais são permitidos, desde que não excedam o valor total

Pseudocódigo para processo de reembolso 
// Função para processar reembolso
async function processarReembolso(pedidoOriginal, valor) {
    // 1. Gerar número único para o reembolso
    const refundNumber = `${pedidoOriginal}ref${Date.now()}`;
    
    // 2. Preparar biz_content
    const bizContent = {
        orig_out_trade_no: pedidoOriginal,
        out_trade_no: refundNumber,
        refund_amount: valor
    };
    
    // 3. Encriptar e enviar requisição
    const resposta = await solicitarReembolso(bizContent);
    
    if (resposta.code === 'S0001') {
        // 4. Aguardar processamento (polling ou webhook)
        const statusFinal = await aguardarConfirmacao(refundNumber);
        
        if (statusFinal === 'S') {
            return { sucesso: true, mensagem: 'Reembolso concluído' };
        } else {
            return { sucesso: false, mensagem: 'Falha no reembolso' };
        }
    }
    
    return { sucesso: false, mensagem: resposta.sub_msg };
}
